1. Preparing

Make sure you have the following:

  • Apache 2 or higher, with:
  • mod_rewrite
  • PHP 5.4 or higher, with:
  • MySQLi
  • cURL
  • OpenSSL
  • GD
  • mbstring
  • MySQL 5 or higher
1. Installation

Do the following to complete the installation process

  • Import the musicEngine.SQL file from the "MySQL" folder into your MySQL database using the import function available in MySQL.
  • With a text editor open the following file located inside the 'Script/' folder: config.php, and replace what's between the quotes (e.g: 'YOURDBNAME') with your MySQL settings:
  • As shown in this image
  • 'db_host' => 'YOURDBHOST',
  • 'db_username' => 'YOURDBUSER',
  • 'db_name' => 'YOURDBNAME',
  • 'db_password' => 'YOURDBPASSWORD',
  • Upload the files from the "Script" folder on the FTP server (usually public_html folder when installing it on a domain).
  • Set the CHMOD to 777, 775 or 775 (depending on the server configuration) to the following folders: '/uploads'.

1. Faqs
  • Can't upload tracks

    Make sure you have set the following php settings to higher value in your php.in

  • memory_limit 656M
  • post_max_size 656M
  • upload_max_filesize 604M
  • max_input_vars 200000
  • max_execution_time 30000
  • max_input_time 600000
    •

    Also you need to increase the default audio size in the admin panel -> settings -> limit

    Disable mod_security

    Disable mod-security if enabled on your server, you can contact your hosting to disable it

    You can also disable it by inserting this in your .htaccess file

    <IfModule mod_security.c>
    SecFilterEngine Off
    </IfModule>
  • Facebook Login Not Working

    Please make sure you have enabled permalinks because non-permalinks cause redirect loop

  • How to Change the Url to Permalinks

    1. To change links to permalinks go to config.php change false to true as shown below

    2. Create an .htaccess file on your root folder and copy this content below into it

    IndexIgnore *
    <IfModule mod_rewrite.c>
    #Options +FollowSymLinks
    RewriteEngine On
    RewriteCond %{REQUEST_FILENAME} !-d
    RewriteCond %{REQUEST_FILENAME} !-f
    RewriteRule ^ index.php [L]
    </IfModule>
    <FilesMatch “.(ttf|otf|eot|woff)$”>
    <IfModule mod_headers.c>
    Header set Access-Control-Allow-Origin “*”
    </IfModule>
    </FilesMatch>

    Make sure mod_rewrite is enabled on your server to prevent any error

  • How to Enable HTTPS

    Go to config.php set https to true as show below

    How to Force HTTP to HTTPS through .htaccess

    If you don't have the .htaccess file please follow the instruction given above at permalinks setup, copy this into your htaccess file right after RewriteEngine On as show in the image below

    RewriteCond %{HTTPS} off
    RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]
